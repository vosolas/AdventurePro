# AdventurePro

**A Revolutionary Blockchain Gaming Platform**

AdventurePro is an innovative decentralized gaming ecosystem that combines competitive gameplay with blockchain technology. Built on Ethereum and powered by Fully Homomorphic Encryption (FHE), it creates a fair, transparent, and secure gaming environment where players can compete, earn rewards, and build their digital legacy.


3. **Configure environment**
